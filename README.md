# wmUpdate
The repository includes codes for: 
- Information decoding
- Neural subspace alignment
- Neural selectivity

## Analyses
### Decoding analysis
The `decoding_analysis` folder contains code for constructing neural pseudopopulations across recording sessions and decoding target locations from spiking activity. 

### State space analsyis
The `state_space_analysis` folder contains code for computing the alignment between target neural subspaces. 

### Selectivity analysis
The `neural_selectivity_analysis` folder contains codes for identifying selective neurons for different targets before and after decision.
